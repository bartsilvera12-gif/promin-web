Cómo publicar los cambios

El panel guarda tus cambios en este navegador. Para que aparezcan en el sitio en vivo (promin-web.vercel.app) hay dos caminos:

  1. Exportar el contenido con Exportar JSON y aplicarlo al sitio.
  2. Conectar las páginas a content.json (lo puede dejar listo Neura): así, al subir el JSON exportado, el sitio se actualiza solo.

Para un panel con login real y base de datos (como una tienda con Supabase), hace falta conectar un backend — Neura puede montarlo.
